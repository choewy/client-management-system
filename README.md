# 고객 관리 시스템

- Git Clone

```
$ git clone https://github.com/choewy/client-management-system.git
```

- Docker Container
- service: MySQL(8.x)
- port : 5001
- db : service

```
$ docker-compose up -d
```

- Install Dependencies

```
$ npm run setting
```

- Run devlopment mode

```
$ npm run start:dev
```

- Run Client only

```
$ npm run client
```

- Run Server only

```
$ npm run server
```
