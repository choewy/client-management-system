# 고객 관리 시스템

## Git Clone

```
$ git clone https://github.com/choewy/client-management-system.git
```

## Docker Container

```
$ docker-compose up -d
```

- service: MySQL(8.x)
- port : 5001
- db : service

## Install Dependencies

```
$ npm run setting
```

## Run devlopment mode

### Client & Server

```
$ npm run start:dev
```

### Client only

```
$ npm run client
```

### Server only

```
$ npm run server
```
